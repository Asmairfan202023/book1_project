# Quickstart: Author Guidelines

This document provides guidelines for authors contributing to the "Physical AI & Humanoid Robotics" textbook chapter.

## Writing Style
- **Tone**: Academic, clear, and concise.
- **Audience**: Undergraduate computer science students.
- **Voice**: Active voice is preferred.

## Formatting
- **File Format**: Markdown
- **Citations**: APA 7th edition. Use a citation manager like Zotero or Mendeley.
- **Headings**: Use appropriate markdown heading levels (e.g., `#`, `##`, `###`).

## Docusaurus Setup
1.  **Install Node.js**: Make sure you have Node.js (v16.14 or later) installed.
2.  **Install Docusaurus**: Open a terminal in the project root and run `npm install`.
3.  **Run Docusaurus**: Run `npm start` to start the local development server.
4.  **Content**: The main content file is located at `docs/physical-ai-chapter.md`.

## Workflow
1.  **Research**: Gather sources and add them to `research.md`.
2.  **Outline**: Flesh out the `content_outline.md` with more detail.
3.  **Draft**: Write the chapter content in `docs/physical-ai-chapter.md`.
4.  **Review**: Follow the review process outlined in `plan.md`.
